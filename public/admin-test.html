Headings

## Top level heading - H2
### Second level heading - H3
#### Third level heading - H4

Don't use a single # - this is H1 and is for the title only

Links

Internal links

Policies, publications, news, speeches, detailed guides etc. can be linked using URLs from within the admin interface

[link text](/government/admin/policies/3373 "Hover text goes here")

Series pages, topics, organisations, people, roles etc. should use public URLs from gov.uk

[link text](https://www.gov.uk/government/topics/climate-change "Hover text goes here")

External links

Links with external URLs will automatically display as external links

[link text](http://www.example.com "Hover text goes here")

Media

Images

First upload your image(s), then:

!!n

e.g. for the first image:

!!1

Video links

Links to YouTube videos will automatically be displayed in an embedded media player.

[link text](youtube-video-url)

Use the title of the video for the "link text", e.g. :-

[The Queens Diamond Jubilee Concert](http://www.youtube.com/watch?v=OXHPWmnycno)

Attachments

Not used on publications or consultations (attachments are placed automatically).

First upload your attachment(s), then, to create an attachment callout box:

!@n

Or, if you want to use a link to an attachment in a sentence or list:

[InlineAttachment:n]

e.g. for the first file:

!@1

or:

[InlineAttachment:1]
